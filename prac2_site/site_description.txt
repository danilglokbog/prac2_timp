Данный веб-сайт представляет собой информационную панель "Industrial Safety 360", предназначенную для мониторинга и управления промышленной безопасностью на предприятии. Приложение позволяет сотрудникам и руководству публиковать и изучать общие правила безопасности, вести учет зарегистрированных инцидентов с классификацией по их серьезности, а также планировать и отслеживать предстоящие обучающие тренинги, используя удобную встроенную панель администратора (с эмуляцией базы данных в оперативной памяти).

Описание страниц:
1. index.html (Главная) — отображает сводный дашборд с актуальными правилами, инцидентами и тренингами для всех посетителей.
2. admin.html (Админ-панель) — служит главным интерфейсом для администратора в виде таблиц со всеми данными и кнопками действий.
3. add_rule.html — предоставляет форму для создания и добавления нового правила безопасности.
4. edit_rule.html — используется для изменения текста, названия или категории уже существующего правила.
5. delete_rule.html — запрашивает подтверждение перед удалением выбранного правила из базы.
6. add_incident.html — содержит форму для регистрации нового происшествия или нарушения на производстве.
7. edit_incident.html — позволяет обновить детали, дату, место или уровень серьезности существующего инцидента.
8. delete_incident.html — служит для подтверждения удаления записи об инциденте из системы.
9. add_training.html — страница с формой для назначения нового обучающего тренинга с указанием даты и числа участников.
10. edit_training.html — предназначена для корректировки названия, сроков или вместимости запланированного тренинга.
11. delete_training.html — защищает от случайных действий, требуя подтверждения перед отменой (удалением) тренинга.
